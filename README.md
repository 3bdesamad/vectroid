# Vectroid ✦

> **Offline-first, single-page SVG to Android VectorDrawable XML Converter**

Vectroid is a simple, client-side tool that converts SVG files into Android VectorDrawable XML. It runs entirely inside the web browser with zero server uploads, ensuring complete privacy.

---

## 🚀 How to Run Locally

No installations, build steps, or localhost servers are required.

1. Clone or download this repository.
2. Double-click `index.html` to open it in Google Chrome, Microsoft Edge, Firefox, Safari, or any modern web browser.
3. Drag your `.svg` files onto the drop zone to begin.

---

## 🛠️ Technology Stack
* **HTML5:** Semantic workspace layout.
* **CSS3:** Flexible flexbox grid, Notion-inspired CSS custom properties (variables), theme mode animations, and 6px border-radius styling.
* **Vanilla JavaScript (ES6+):** In-memory SVG rendering, Canvas color translation, and regex-free XML construction.
